---
title: Enterprise License File
description: Siriusec enterprise license file configuration parameters and requirements
---

Commercial Siriusec subscriptions require a valid license. The license file can
be downloaded from the [Siriusec Customer
Portal](https://dashboard.siriusec.com/web/login).

The Siriusec license file contains a X.509 certificate and the corresponding private key in PEM format. Place the downloaded file on Auth servers and set the `license_file` configuration parameter of your `siriusec.yaml` to point to the file location:

```yaml
auth_service:
    license_file: /var/lib/siriusec/license.pem
```

The `license_file` path can be either absolute or relative to the configured
`data_dir`. If the license file path is not set, Siriusec will look for the
`license.pem` file in the configured `data_dir`.

<Admonition
  type="tip"
  title="NOTE"
>
  Only Auth servers require the license. Proxies and Nodes that do not also have Auth role enabled do not need the license.
</Admonition>
