八、快速开始
================================================================================

1. 编译项目:
   $ make clean
   $ make full

2. 验证二进制:
   $ ./build/siriusec version
   $ ./build/sctl version

3. 启动集群:
   $ sudo mkdir -p -m0700 /var/lib/siriusec
   $ sudo chown $USER /var/lib/siriusec
   $ ./build/siriusec start

4. 访问 Web UI:
   浏览器打开: https://localhost:3080

5. 使用 CLI:
   $ ./build/tsh login --proxy=localhost:3080 --user=admin
   $ ./build/tsh ls
