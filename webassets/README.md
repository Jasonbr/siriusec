# Webassets

This repository holds the assets needed to run the web interfaces of our open-source products, such as [Siriusec](https://github.com/siriusec/siriusec) or [Gravity](https://github.com/siriusec/gravity).

The source for those assets is available in the [webapps repository](https://github.com/siriusec/webapps). Issues or requests should be filed in that repository too - this one only exists for distribution purposes and has little manual activity.
